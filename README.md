# Trackcoders Backend

Trackcoders is a platform designed to help users track and compare their coding progress across competitive programming platforms, fostering a collaborative and competitive learning environment. This repository contains the backend implementation of the project.

## Features

- **User Management**:  
  Manage user accounts, including registration, login, and authentication.  

- **Friend System**:  
  Add friends, compare coding progress, and foster collaboration.  

- **Leaderboard**:  
  View a leaderboard to track progress and motivate users with healthy competition.  

- **Web Scraping**:  
  Aggregate coding data from platforms like LeetCode, CodeChef, and GeeksforGeeks.  

- **Efficient Data Storage**:  
  Utilize a scalable database design to manage user profiles, coding progress, and question lists with minimal redundancy.  

- **Browser Extension Integration**:  
  Sync coding data via a browser extension for a seamless user experience.  

## Tech Stack

- **Programming Language**: JavaScript (Node.js)  
- **Framework**: Express.js  
- **Database**: MongoDB  
- **Web Scraping**: Playwright
